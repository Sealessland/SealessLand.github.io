Class

class 定义了一种可以实例化的规则，可以创造同一个class的多个实例。
class的定义方式如下

通过new生成一个实例并调用

class之间可以使用extends进行继承

其中 override关键字表示在子类中重写父类的方法
每个使用new实例化的class之间并不会相互影响
在类的实现中可以定义尚未实现的方法 ，使用关键字 abstract class
